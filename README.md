# Website Portofolio Mahasiswa Sistem Informasi

## Deskripsi Project
Website Portofolio ini merupakan website statis yang dibuat sebagai tugas mata kuliah pemrograman web. Website ini bertujuan untuk menampilkan informasi singkat mengenai profil mahasiswa, kemampuan (skills), pengalaman, serta sertifikat yang dimiliki. Website dirancang dengan tampilan sederhana, rapi, dan responsif.

## Fitur dan Tampilan Website

### 1. Navbar
Navbar berfungsi sebagai navigasi utama untuk berpindah antar section pada website, yaitu Home, About Me, dan Certificates. Navbar dibuat menggunakan Bootstrap sehingga bersifat responsif dan dapat menyesuaikan dengan ukuran layar perangkat.

### 2. Home (Hero Section)
Section Home berisi:
- Foto profil
- Nama
- Status sebagai Mahasiswa Sistem Informasi

Bagian ini digunakan sebagai perkenalan singkat kepada pengunjung website.

### 3. About Me
Section About Me menampilkan:
- Deskripsi singkat mengenai diri
- Daftar pengalaman yang relevan
- Daftar skills yang ditampilkan menggunakan progress bar

Progress bar digunakan untuk menunjukkan tingkat penguasaan kemampuan seperti HTML, CSS, Bootstrap, dan dasar basis data.

### 4. Certificates
Section Certificates menampilkan daftar sertifikat dalam bentuk card. Setiap card berisi judul dan deskripsi singkat sertifikat. Layout disusun menggunakan grid system agar tampilan lebih rapi dan responsif.

## Penjelasan Kode

### index.html
- Menggunakan struktur dasar HTML (html, head, body)
- Menggunakan Bootstrap 5 melalui CDN
- Berisi struktur utama website seperti navbar, Home, About Me, Certificates, dan footer

### style.css
- Digunakan untuk mengatur tampilan tambahan seperti warna, background, font, dan animasi
- Mengatur ukuran dan bentuk gambar profil
- Menambahkan efek hover pada card sertifikat
